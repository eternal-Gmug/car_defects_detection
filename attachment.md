# 灰度图、彩色图、二值图的理解
- 灰度值是指图像中每个像素的亮度与灰度级别。在数字图像中，灰度值通常表示像素的亮度强度，灰度值越高，像素越亮，灰度值越低，像素越暗，0表示最暗的黑色，255表示最亮的白色。
- 灰度图是指将彩色图像转换成灰度图像的结果。每个像素只有一个灰度值，该灰度值表示原始图像中该像素的亮度强度。
- 彩色图是指每个像素点的取值由RGB三个分量组成，每个分量的取值通常为0到255，每个像素点的RGB分量的取值决定了该像素的颜色。
- 二值图是指像素的取值只有两种可能的图像。在二值图中，可以认为目标的区域通常为白色，背景通常为黑色。
## 灰色图像与彩色图像的转换     
- 平均值法（应用最广泛）：G = (R + G + B) / 3
- 加权平均法：G = 0.299R + 0.587G + 0.114B
- 使用其中一个彩色通道作为灰度图像，通常选用绿色通道，绿色通道对亮度更敏感。

# 图像增强方法
## 1、同态滤波
- 用于同时校正图像的非均匀光照和增强细节对比度
- 核心在于将图像的光照分量（高频）和细节分量（低频）分离开来，分别进行处理
- 适合应用在处理光照不均匀的图像，如夜间拍摄的图像、曝光不足的图像等
## 2、直方图均衡化
- 直方图均衡化就是把一个已知灰度概率密度分布的图像经过变换，使之演变成一副具有**均匀灰度概率密度分布**的新图像。
## 3、对比度限制的自适应直方图均衡化（CLAHE）
- 作用：增强图像的对比度的同时可以抑制噪声
- 直方图均衡化在处理带有大块暗区或者亮区的图像时，可能会造成灰度集中的区域被过度拉伸，造成噪点
- 加入对比度限制（CLAHE），将直方图超过阈值的分布均匀分散在概率密度分布上，避免某些集中区域被拉得过于拉伸
- 自适应均衡化的基本思想是将原始图像分成子区域，然后对每个子区域独立进行直方图均衡化处理，针对不同区域之间的不连续，
采用双线性插值的AHE
- 适合处理具有全部灰度级且均匀分布的图像，对于有大片过亮或过暗的区域，其效果较差，但CLAHE在一定程度上解决这一问题

# 模型训练指标
- 基本信息指标：
    - epoch: 当前训练轮次
    - time: 训练耗时
- 训练损失指标：
    - train/box_loss: 边界框回归损失
    - train/seg_loss: 分割损失，衡量预测分割掩码与真实掩码的差异
    - train/cls_loss: 分类损失，衡量模型对目标类别的分类能力
    - train/dfl_loss: 分布焦点损失，用于处理类别不平衡问题，用于边界框坐标预测的辅助损失
- 检测指标：
    - metrics/precision(B): 检测精确率，正确预测为正类的样本占所有预测为正类的样本比例
    - metrics/recall(B): 检测召回率，正确预测为正类的样本占所有实际为正类的样本比例
    - metrics/mAP50(B): 检测mAP50，IoU阈值为0.5时的平均精度
    - metrics/mAP50-95(B): 检测mAP50-95，IoU阈值从0.5到0.95，每隔0.05取一个值，计算平均精度
- 分割指标：
    - metrics/precision(M): 分割精确率，正确预测为正类的样本占所有预测为正类的样本比例
    - metrics/recall(M): 分割召回率，正确预测为正类的样本占所有实际为正类的样本比例
    - metrics/mAP50(M): 分割mAP50，IoU阈值为0.5时的平均精度
    - metrics/mAP50-95(M): 分割mAP50-95，IoU阈值从0.5到0.95，每隔0.05取一个值，计算平均精度
- 验证损失指标:
    - val/box_loss: 验证集边界框回归损失
    - val/seg_loss: 验证集分割损失
    - val/cls_loss: 验证集分类损失
    - val/dfl_loss: 验证集分布焦点损失
